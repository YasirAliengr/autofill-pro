AutoFill Pro — Privacy Policy

Last updated: April 13, 2026

Summary: AutoFill Pro stores all data locally on your device. We do not collect, transmit, or share any personal information. No analytics, no tracking, no servers.

1. Data Storage

All data you enter (autofill rules, profiles, variables, text clips, settings) is stored exclusively in your browser's local storage using the chrome.storage.local API. This data never leaves your device.

2. Data Collection

AutoFill Pro does not collect any data. Specifically, we do not collect:

  • Personal information (name, email, address)
  • Browsing history or visited URLs
  • Form data you fill on websites
  • Usage analytics or telemetry
  • Crash reports

3. Website Access

AutoFill Pro requires access to websites (http/https) to function — it needs to read form fields on pages and fill them with your configured rules. The extension:

  • Only reads form field attributes (name, id, placeholder) to match your rules
  • Only writes values you have explicitly configured into matching fields
  • Does not read or store page content, cookies, or passwords from websites
  • Does not inject ads, trackers, or third-party scripts

4. Encryption

Password values can be encrypted using AES-GCM (256-bit) via the Web Crypto API before storage. The encryption key is derived from your browser's unique extension installation ID, ensuring encrypted values cannot be read outside your browser.

5. Third-Party Services

AutoFill Pro does not communicate with any external servers, APIs, or third-party services. The extension works entirely offline after installation.

6. Data Deletion

You can delete all stored data at any time via the "Factory Reset" option in Settings. Uninstalling the extension also removes all stored data from your browser.

7. Permissions Justification

  • storage — Save your autofill rules, profiles, and settings locally
  • activeTab — Access the current tab to fill forms when you trigger autofill
  • scripting — Inject the autofill engine into web pages to read and fill form fields
  • contextMenus — Add right-click menu options (Fill page, Add rules, etc.)
  • sidePanel — Provide a side panel for quick access to profiles and actions
  • host_permissions (http/https) — Required to run the content script on web pages containing forms

8. Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date above. Continued use of the extension after changes constitutes acceptance of the updated policy.

9. Contact

For questions about this privacy policy or the extension, please open an issue on our support page or contact us through the Chrome Web Store listing.
